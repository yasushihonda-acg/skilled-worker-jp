# skilled-worker-jp

特定技能外国人向け人材募集サイト

## 概要

日本で働きたい特定技能外国人と、人材を求める企業をつなぐプラットフォーム。

## 特徴

- 5言語対応（日本語/英語/インドネシア語/ネパール語/ビルマ語）
- AIチャットによる多言語サポート
- シンプルな求職者登録フロー

## 開発環境

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
```

http://localhost:3000 でアクセス

## ビルド

```bash
npm run build
npm start
```
